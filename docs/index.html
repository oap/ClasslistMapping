AWS Academy Canvas & D2L Classlist Mapping

This tool allows you to map AWS Academy class sections to their corresponding Canvas and D2L sections.

  • Download Tampermonkey.
  • Install the Class Mapping User Script.
  • Click the "Add New Mapping" button to add a new mapping.
  • Class Name: The name of the Course your are teaching.
  • Section: The section of the Course you are teaching.
  • Canvas ID: The ID of the Canvas section that corresponds to the AWS Academy class section.
  • D2L ID: The ID of the D2L section that corresponds to the AWS Academy class section.

After you added the mapping, click the links to visit class list pages on both LMS platforms.

The script will get the classlists and save them locally in your browser.

Click the Class Name to see the detailed list of users in your classes.

Click the "Edit" button to update the mapping or the "Delete" button to remove it.
